# 需求文档归档

存放**历次**需求文档，用于版本追溯与评审记录。**只追加、不覆盖**历史文件。

## 目录结构

```
需求/
├── _templates/          # PRD、需求变更单、评审纪要模板
└── 归档/
    ├── 按版本/          # 大里程碑（如 v0.1-MVP）
    └── 按模块/          # 模块迭代（如 ai观测/）
```

## 命名规范

| 场景 | 格式 | 示例 |
|------|------|------|
| 模块迭代 | `{YYYY-MM-DD}-{简述}.md` | `2026-06-11-初版PRD.md` |
| 版本里程碑 | `{序号}-{文档类型}-{主题}.md` | `04-PRD-AI观测.md` |

## 归档流程

1. 需求评审通过或迭代完成后，将定稿复制到 `归档/按模块/{模块}/`
2. 大版本发布时，汇总复制到 `归档/按版本/vX.X/`
3. 当前有效功能说明维护在 `文档/功能列表/`，不在此重复维护
