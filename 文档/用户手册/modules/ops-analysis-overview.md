# 运营分析 — 平台总览

> 原型：`原型/运营/运营分析/index.html`

## 入口

Sidebar → **运营** → **运营分析**

## 看板 Tab

| Tab | 用途 |
|-----|------|
| 平台总览 | 跨域 KPI、健康度、需关注事项、AI 智能运营 |
| 使用分析 | 用户活跃、Adoption 漏斗 |
| 资产效能 | Agent/Skill/MCP 调用与闲置 |
| 成本效率 | 成本与 Model 分布 |
| 审批发布 | 审批积压与发布记录 |
| 集市运营 | Skillhub/Agenthub 供给消费 |
| 质量反馈 | 错误率与用户反馈 |

## 工具栏

- **Workspace**：All 或单个 Workspace，指标按范围缩放
- **时间范围**：7d / 30d / 90d
- **导出周报**（平台总览 Tab）：生成 Markdown 预览

## AI 智能运营

在「平台总览」顶部：

1. 进入即可阅读默认运营洞察
2. 输入中文问题点击「智能分析」
3. 点击「需关注」表格行查看 AI 解读

其他 Tab 右下角 **✦** 按钮可打开看板内追问。

## 明细排查

运营分析只做聚合；Trace、审计、审批操作请跳转：

- AI观测（成本/错误 Trace）
- AI治理（风险告警）
- 审批中心（待审列表）
