# AI 观测 — Tracing 使用说明

## 进入路径

Sidebar → **运营** → **AI观测** → **Tracing** 标签页

## 筛选 Trace

1. 点击 **Show filters** 展开左侧 Filters 面板
2. 勾选 Type、Status 等条件，或输入 Trace Name / User ID
3. 点击 **AI 筛选**，用中文描述条件（如「最近错误的 tool 调用」），回车应用

## 查看详情

点击表格任意行，右侧打开 Trace 详情抽屉：

- 左侧：Observation 树、Topology 拓扑
- 右侧：Preview / Scores / Log、Input / Output

点击遮罩或关闭按钮退出详情。

## Workspace 过滤

工具栏 **Workspace** 下拉选择与 Sidebar 顶部一致的 Workspace，列表仅显示该空间数据。
