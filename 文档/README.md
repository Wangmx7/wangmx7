# 产品文档（当前有效版本）

本目录存放**持续更新**的产品文档，与 `需求/归档/` 中的历史需求文档区分。

## 子目录

| 路径 | 内容 |
|------|------|
| [`功能列表/`](功能列表/) | 全平台功能清单（`index.json` + 模块 Markdown） |
| [`用户手册/`](用户手册/) | 面向终端用户/管理员的操作说明 |

## 维护规范

- 功能变更时：**先**更新 `功能列表/index.json`，**再**改原型
- 用户手册截图与说明应对齐 `原型/` 中对应页面
- 不在此目录存放 PRD 初稿；PRD 归档至 `需求/归档/`
- 禁止修改 `需求/归档/` 中已归档的历史文件

## 与 sidebar-config 对齐

`共享/sidebar/sidebar-config.js` 中每项的 `featureModule` / `manualModule` 对应本目录下 `modules/` 中的文件名（不含扩展名）。
