# 功能列表概览

## 原型状态

| 状态 | 含义 |
|------|------|
| `ready` | 已有可交互 HTML 原型 |
| `wip` | 原型开发中 |
| `placeholder` | 仅占位说明 |

- **AI观测** — `原型/运营/ai观测/`，详见 [`modules/ai-observability.md`](modules/ai-observability.md)
- **运营分析** — `原型/运营/运营分析/`，详见 [`modules/ops-analysis.md`](modules/ops-analysis.md)

## 维护

1. `共享/sidebar/sidebar-config.js`
2. 本目录 `index.json`
3. 可选 `modules/{id}.md`
